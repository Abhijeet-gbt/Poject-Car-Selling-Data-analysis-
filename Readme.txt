# 🚗 Car Sales Data Analysis (Python Project)

## 📌 Project Overview
This project performs data analysis on car sales data using Python, Pandas, and Matplotlib.  
It explores pricing trends, seller contributions, fuel types, and transmission distribution.

The main goal is to extract meaningful insights and visualize them using graphs.

---

## 📂 Dataset
- File used: `car_details_raw_data.csv`
- Dataset includes:
  - Car name
  - Year of manufacture
  - Selling price
  - Fuel type
  - Seller type
  - Transmission (Manual/Automatic)

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas 📊
- Matplotlib 📈

---

## 🔍 Key Operations Performed

### 1. Data Understanding
- Display dataset
- View first 10 rows
- Check dataset structure
- Summary statistics

### 2. Data Cleaning
- Checked null values
- Checked duplicate values
- Removed duplicates

### 3. Data Analysis
- Sorted cars by selling price
- Filtered cars by year and price range
- Average selling price by seller type
- Transmission distribution (Manual vs Automatic)
- Fuel type distribution

### 4. Feature Engineering
- Extracted car brand from name
- Top 5 brands by average selling price

---

## 📊 Visualizations

This project creates a 4-in-1 subplot:

1. Manual vs Automatic Cars (Bar Chart)
2. Seller Contribution (Pie Chart)
3. Top 5 Brands by Average Selling Price (Bar Chart)
4. Fuel Type Distribution (Bar Chart)

Output file:
Car_Sell_DataAnalysis.png

- High resolution (600 DPI)
- Ready for reports and presentations
